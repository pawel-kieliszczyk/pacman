# Pacman

**Requirements**
 + OS X
 + Clang 3.4
 + Boost
 
**Download**
```
git clone https://github.com/pawel-kieliszczyk/pacman.git
```

**Compilation**
```
cd Build
scons -f SConstruct-application
```
